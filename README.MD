# Virtual Poker

Simple virtual poker game using console UI.

### Prize table:

| Hand | Prize |
| --- | --- |
| Royal Flush | 800 |
| Straight Flush | 50 |
| Four of a kind | 25 |
| Full House | 9 |
| Flush | 6 |
| Straight | 4 |
| Three of a kind | 3 |
| Two Pair | 2 |
| Jacks or Better | 1 |
| All other | 0 |

### Running the game:

    Navigate to "src" folder and execute following commands:
        javac Game.java
        java Game
